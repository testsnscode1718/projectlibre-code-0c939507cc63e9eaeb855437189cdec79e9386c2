ProjectLibre is the updated OpenProj.
The major improvements includes:
- Compatibility with Microsoft Project 2010, 2013, 2016
- Major bug fixes and more

Windows notes:
32 bits versions: projectlibre-@version@.msi
64 bits versions: projectlibre-@version@-x64.msi

Mac OS X notes:
ProjectLibre requires Mac OS X 10.5.7 or later.
- Versions 10.7.3+, 10.8, 10.9, 10.10, 10.11, 10.12  (64bits) with or without Java already installed: download projectlibre-@version@.dmg. It comes with Java embedded.
- Versions 10.7.3+, 10.8, 10.9, 10.10, 10.11, 10.12 with java7 already installed: download projectlibre-@version@.dmg (64bits only) or projectlibre-java7-@version@.dmg (32bits or 64bits). 
- Versions <=10.7.2: download projectlibre-java6-@version@.dmg.
Note for 10.5: You have to update to 10.5.7+. Make sure java updates have been installed (http://support.apple.com/kb/DL848). Then select Java 6 as default in /Applications/Utility/Java Preferences.

Linux notes:
- Fedora/RedHat Enterprise/Oracle/Centos, openSUSE/SLES/SLED, Mageia/Mandriva/PCLinuxOS: download projectlibre-@rpm_version@-@rpm_revision@.rpm
- Ubuntu/Mint/Debian: download projectlibre-@deb_version@-@deb_revision@.deb
- Slackware and distributions not using rpm or deb: download projectlibre-@version@.tar.gz
